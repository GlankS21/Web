## Command Line


### 1. Создание repository Repository
Добавляние нового удаленного репозиторий к локальному проекту
```bash
git remote add origin https://github.com/GlankS21/web-porfolio.git
# проверка remote
git remote -v
```
### 2. Добавление и сохранение истории
Добавляние всех новых, измененных и удаленных файлы из текущей директории и её поддиректорий в область подготовки
```bash
git add . 
git commit -m "first commit"
```
### 3. Отправка изменений на GitHub
Отправка недавних коммитов c компьютера локального репозитория на сервер с удаленным репозиторием
```bash
git push -u origin main 
```
### 4. Конечное состояние репозитория
Выведение коммит в виде одной строки при выполнении команды git log с этой опцией
```bash
git log --oneline
```
